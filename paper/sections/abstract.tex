\begin{abstract}
Low-visibility underwater scenes contain severe color distortion, scattering artifacts,
and weak structural details, which make small biological object detection difficult.
To address this issue, we propose a quality-aware dual-branch cross-residual network
(QDCR-Net) that jointly models raw-image semantics and enhancement-guided structural
details. The proposed framework contains a raw branch, an enhanced branch, a bidirectional
cross-residual interaction module, a quality-aware dynamic fusion module, and a
small-object-oriented detection neck. The design aims to retain reliable semantic cues
from the original image while adaptively exploiting enhancement benefits under different
degradation conditions. We plan to evaluate the method on URPC2018 and RUOD using
accuracy, complexity, and runtime metrics, together with ablation and robustness studies.
\end{abstract}

\noindent\textbf{Keywords:} underwater object detection, small object detection,
dual-branch network, quality-aware fusion, low-visibility scenes
